Literature

Reference papers

  1. Granmo, O.C., 2018. The Tsetlin Machine–A Game Theoretic Bandit Driven Approach to Optimal Pattern Recognition with Propositional Logic. arXiv preprint arXiv:1804.01508.

    Original paper which introduces Tsetlin Machines Classifier.

  2. Abeyrathna, K.D., Granmo, O.C., Zhang, X. and Goodwin, M., 2019, July. A scheme for continuous input to the Tsetlin machine with applications to forecasting disease outbreaks. In International conference on industrial, engineering and other applications of applied intelligent systems (pp. 564-578). Springer, Cham. arXiv preprint arXiv:1905.04199.

    Tsetlin Machine with continuous features.

  3. Abeyrathna, K.D., Granmo, O.C., Jiao, L. and Goodwin, M., 2019, September. The regression Tsetlin machine: a Tsetlin machine for continuous output problems. In EPIA Conference on Artificial Intelligence (pp. 268-280). Springer, Cham. arXiv preprint arXiv:1905.04206.

    Paper which introduces Tsetlin Machine Regressor.

  4. Abeyrathna, K.D., Granmo, O.C. and Goodwin, M., 2020. A regression Tsetlin Machine with integer weighted clauses for compact pattern representation. arXiv preprint arXiv:2002.01245.

    Tsetlin Machine with integer-weighted clauses.
